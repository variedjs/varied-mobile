### 特性

- 完善的文档和示例
- 支持按需引入
- 支持 TS

### 安装

```bash
# 通过 npm 安装
npm i @varied/mobile -S

# 通过 yarn 安装
yarn add @varied/mobile
```

### 快速上手

```js
import Vue from "vue";
import { Button } from "@varied/mobile";
import "@varied/mobile/lib/index.less";

Vue.use(Button);
```

### 浏览器支持

现代浏览器以及 Android 4.0+, iOS 7+
